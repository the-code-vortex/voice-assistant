# Voice Assistant

A Python voice assistant with OpenAI integration.

## Features
- Speech recognition
- OpenAI GPT responses
- Wikipedia search
- Web browser control
- Time and date telling

## Installation
1. Install dependencies: \pip install -r requirements.txt\`n2. Get OpenAI API key from: https://platform.openai.com/api-keys
3. Set environment variable: \export OPENAI_API_KEY=your_key_here\`n4. Run: \python voice_assistant.py\`n
## Commands
- 'Open YouTube' - Opens YouTube
- 'Search Wikipedia for [topic]' - Wikipedia search
- 'What time is it?' - Tells current time
- 'Ask OpenAI [question]' - Gets GPT response
- 'Exit' - Quits program
